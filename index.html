Viewport

The viewport is the area of the window in which web content can be seen. It typically refers to the visible portion of a web page in a browser, excluding elements like toolbars and scroll bars.

Flexbox

Flexbox is a CSS layout model that allows items within a container to be automatically arranged and aligned along one or both axes, providing flexible and efficient ways to distribute space.

Grid

The CSS Grid Layout is a two-dimensional layout system for the web that enables developers to create complex grid-based designs with rows and columns, making it easier to build responsive layouts.
