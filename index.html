Viewport

The viewport is the area of the window in which the content of the webpage can be seen. It refers to the visible parts of a web page, excluding elements like toolbars and scroll bars.

Flexbox

Flexbox is a CSS layout that allows items to be automatically arranged and aligned, allowing for quality and efficient distributiond of space.

Grid

The CSS Grid is a two-dimensional layout for webpages that allows developers to create complex grid designs making it easier to build a layout that is responsive.
